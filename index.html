Course goal

This training program is designed to guide you through the fundamental aspects of data analysis. You'll explore various databases, learn SQL, dive into data modeling, and use modern tools like Python for data analysis and visualization.

Course Content

  • 1. Introduction to Data Analysis ver más
    • Basics of data analysis
    • Importance in the current landscape
    • Common tools and applications
  • 2. Databases and Set Theory ver más
    • Database types and uses
    • Set theory and databases
    • Basic set theory operations in databases
  • 3. Data Modeling
    • Entity-relationship models
    • Data normalization
    • Logical and physical database design
  • 4. SQL and Data Manipulation
    • SQL fundamentals: queries, inserts, updates, deletes
    • Aggregate functions and grouping
    • Joins and subqueries
  • 5. Data Analysis with Python
    • Introduction to Python for data analysis
    • Python libraries: Pandas, NumPy, Matplotlib
    • Exploratory data analysis
  • 6. Data Visualization
    • Data visualization principles
    • Visualization tools: Tableau, Power BI, Python libraries
    • Interactive dashboards
  • 7. Machine Learning for Data Analysis
    • Introduction to machine learning
    • Regression and classification models
    • Model evaluation and hyperparameter tuning
  • 8. Big Data and Associated Technologies
    • Big Data concepts
    • Introduction to Hadoop and Spark
    • Data storage and processing at scale
  • 9. Practical Projects and Case Studies
    • Real project development
    • Case study analysis
    • Best practices and lessons learned
